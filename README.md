# Livskin RevOps Stack

Infraestructura base del proyecto Livskin VPS Operaciones.

## Base actual
- VPS: DigitalOcean
- OS: Ubuntu 22.04 LTS
- Reverse proxy: Nginx
- Contenedores: Docker
- Automatización: n8n
- CRM: Vtiger
- Base analítica: PostgreSQL
- Dashboards: Metabase

## Objetivo
Construir una arquitectura RevOps personal-profesional reproducible, versionada y documentada.
