# Brand System — Doctrina maestra de creación de contenido Livskin

> **v0.1 BORRADOR** (modo bootstrap principio #13). Refinamientos esperados de la primera campaña.

---

## 1. Principio fundamental — vinculante absoluto

### ❌ NO vendemos tratamientos
### ✅ Vendemos decisiones sobre identidad

En medicina estética, **3 errores comunes** que NO cometemos:

1. Hablar de **producto** ("botox", "ácido hialurónico")
2. Hablar de **resultado** ("rejuvenecer", "eliminar arrugas")
3. Hablar de **precio o promoción**

Esos errores generan: ansiedad, comparación, desconfianza, sensación de venta barata.

### Enfoque correcto — secuencia obligatoria

```
1. Activar IDENTIDAD
2. Activar DECISIÓN
3. Activar CONTROL personal
4. SOLO después: introducir tratamiento como herramienta (consecuencia de la decisión)
```

## 2. Base teórica aplicada

### 2.1 Decoded — Phil Barden

**Principio aplicado: las personas no compran productos, compran significados.**

| ❌ Lo que NO vendemos | ✅ Lo que SÍ vendemos |
|---|---|
| "Armonización facial" | "Decidir sobre tu rostro sin permiso" |
| "Botox que rejuvenece" | "Cómo quieres que se vea tu armonía" |

**Principio aplicado: el cerebro responde a códigos, no a argumentos.**

Códigos rectores de Livskin:
- **Autonomía** (referencia 8M / decisión personal)
- **Seguridad** (criterio médico, no comercial)
- **Naturalidad**
- **No exageración**
- **Control personal**

**Principio aplicado: imagen > texto.**
- Elegimos rostros que YA transmiten el mensaje
- El copy SOLO refuerza, no carga el peso comunicativo

### 2.2 Nudge — Thaler & Sunstein

**Principio aplicado: no empujar, guiar.**

| ❌ Empuje | ✅ Guía |
|---|---|
| "Agenda ahora" | "Descubre más →" |
| "¡Aprovecha hoy!" | "Conoce tu enfoque" |
| "Última oportunidad" | (NO usar — genera ansiedad) |

**Principio aplicado: reducir fricción.**
- CTA simple, único por sección
- NO múltiples mensajes simultáneos
- NO saturación visual

**Principio aplicado: arquitectura de decisión.**
- TOFU → MOFU → BOFU
- Cada pieza tiene UN rol claro, NO mezcla funciones

### 2.3 Predictably Irrational — Dan Ariely

**Principio aplicado: menos opciones = más decisión.**
- NO listamos tratamientos en TOFU
- NO saturamos con beneficios

**Principio aplicado: el contexto define percepción.**
- Diseño editorial → percepción premium
- Espacio en blanco → valor

**Principio aplicado: el exceso de información bloquea acción.**
- Eliminamos texto innecesario
- Simplificamos mensajes

## 3. Estructura del funnel

### 3.1 TOFU — Atracción

**Objetivo:** generar conexión emocional. **NO vender.**

| Característica | Regla |
|---|---|
| Mencionar tratamientos | ❌ Prohibido |
| Mencionar precios | ❌ Prohibido |
| Hablar de problemas físicos | ❌ Prohibido |
| Generar inseguridad | ❌ Prohibido |
| Tipo de mensaje | Declaración de identidad |
| Imagen | Mujer real, seguridad tranquila, NO modelo artificial |
| CTA | Suave, editorial ("Descubre más →") |

**Ejemplos canónicos:**
- *"Tu rostro no necesita permiso"*
- *"Este 8M, decide por ti"*
- *"Este Día de la Madre, decide por ti"*

### 3.2 MOFU — Consideración

**Objetivo:** introducir el concepto de tratamiento sin romper la emoción.

**Error común que NO cometemos:** *"Ácido hialurónico · Botox · Exosomas"* directo (rompe narrativa).

**Enfoque correcto:** el tratamiento es **consecuencia de la decisión**.

| Característica | Regla |
|---|---|
| Tipo de mensaje | Explicativo emocional |
| Imagen | Mismo estilo que TOFU (coherencia visual obligatoria) |
| CTA | Evoluciona, NO repite ("Conoce tu enfoque", "Explora tu armonización") |

**Ejemplos canónicos:**
- *"La armonización es una decisión personalizada"*
- *"Cada rostro tiene su propia forma de armonizar"*

### 3.3 BOFU — Conversión

**Objetivo:** acción concreta.

**Aquí SÍ se permite:**
- ✅ Hablar de evaluación
- ✅ Hablar de consulta médica
- ✅ Mencionar el procedimiento (con criterio)

| Característica | Regla |
|---|---|
| Tipo de mensaje | Directo pero elegante |
| Imagen | Más cercana, más directa, puede mostrar resultados sutiles |
| CTA | Claro |

**Ejemplos canónicos:**
- *"Evalúa tu combinación ideal"*
- *"Agenda tu evaluación"*

## 4. Errores que evitamos (lista negra)

| Categoría | Ejemplo prohibido | Por qué |
|---|---|---|
| Campaña tipo clínica | "Botox desde S/.X", "Elimina arrugas", "Resultados inmediatos" | Suena a venta barata |
| Saturación | Titular + subtítulo + 5 beneficios + lista + CTA en una sección | Bloquea acción |
| Ruptura de narrativa | TOFU emocional → MOFU técnico frío | Pierde la conexión activada |
| Estética barata | Botones grandes, colores agresivos, tipografía genérica | Rompe el código premium |

---

## 5. Filosofía final de la marca

| Livskin NO vende | Livskin SÍ vende |
|---|---|
| Tratamientos | **Decisión** |
| Resultados específicos | **Control** |
| Promociones | **Naturalidad** |
| Velocidad | **Seguridad** |

**Traducción operativa:** el tratamiento NO es el protagonista. **La persona lo es.**

## 6. Checklist de 4 preguntas — gate obligatorio para cada campaña

Antes de aprobar cualquier campaña / ad / landing, responder explícitamente:

```
[ ] 1. ¿Qué identidad activa?
[ ] 2. ¿Qué emoción genera?
[ ] 3. ¿Qué decisión sugiere?
[ ] 4. ¿Qué NO está diciendo?
```

**Si las 4 están respondidas con claridad → la campaña está bien construida.**

Si alguna está ambigua → no se publica hasta resolverla.

Plantilla a llenar: [campaign-brief-template.md](campaign-brief-template.md).
