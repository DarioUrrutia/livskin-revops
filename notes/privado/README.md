# notes/privado/ — Tu espacio personal

Este directorio está **gitignored**. Todo lo que pongas aquí se queda en tu laptop, no va a GitHub.

Úsalo para:
- Pensamientos libres
- Dudas que no sabes aún cómo formular
- Ideas a medio cocinar
- Notas de referencia personales
- Recordatorios

Cuando una nota aquí madura y vale la pena compartirla con Claude Code / integrar al proyecto, muévela a `notes/compartido/` o pídemela que la integre a docs formales.

**Este README se versiona para que la estructura exista, pero su contenido es solo explicativo.**
